# Warp Infinite 用户使用指南

## 📖 软件简介

Warp Infinite 是一个专为解决 Warp 应用 ，让您的 Warp 应用能够正常连接和使用。
[![Telegram](https://img.shields.io/badge/Telegram-加入群组-blue?style=for-the-badge&logo=telegram)](https://t.me/+HIMW7D6lcfQ0ZDkx)
## ✨ 主要功能

- 🔧 **自动证书管理** - 程序会自动生成和安装所需的 SSL 证书
- 🎯 **智能代理** - 仅代理必要的网络请求，不影响其他应用
- 🎛️ **自定义** - 支持自定义，灵活配置服务器
- 🖥️ **简洁界面** - 直观的图形界面，操作简单
- 🔒 **安全可靠** - 本地运行，不收集任何个人数据

## 🚀 快速开始

### 第一步：下载软件
1. 前往 GitHub Releases 页面
2. 根据您的操作系统下载对应版本：
   - **Windows**: `warp-infinite-windows-final.exe` (密码: `123`)
   - **macOS**: `warp-infinite-macos-final`
   - **Linux**: `warp-infinite-linux-final`
3. 将文件保存到您希望的位置

### 📋 macOS 用户特别说明
**重要：macOS 用户需要手动安装证书**

下载并运行程序后，需要执行以下命令安装证书：

```bash
sudo security add-trusted-cert -d -r trustRoot -k /Library/Keychains/System.keychain ~/certs/mitmproxy-ca-cert.pem
```

**命令说明：**
- 此命令将程序生成的 CA 证书添加到系统信任列表
- 需要管理员权限（sudo）
- 证书路径：`~/certs/mitmproxy-ca-cert.pem`
- 执行前请确保程序已运行并生成了证书文件
### 第二步：准备工作
在使用本软件之前，请确保：

1. **关闭所有 VPN 软件**
   - 关闭任何正在运行的 VPN 应用
   - 包括但不限于：ExpressVPN、NordVPN、Surfshark 等
   - 确保网络环境干净

2. **确保 Warp 应用已安装**
   - 从官方渠道下载并安装  Warp
   - 记住 Warp 的安装路径

### 第三步：运行软件
1. **Windows**: 双击运行 `warp-infinite-windows-final.exe`
2. **macOS**: 在终端中运行 `./warp-infinite-macos-final`
3. **Linux**: 在终端中运行 `./warp-infinite-linux-final`

**首次运行注意事项：**
- **Windows**: 可能显示安全警告，点击"更多信息"→"仍要运行"
- **macOS**: 可能需要在"安全性与隐私"中允许运行未签名应用
- **Linux**: 确保文件具有执行权限 `chmod +x warp-infinite-linux-final`
## 🎮 使用步骤

### 1. 选择 Warp 路径
- 点击"选择 Warp 路径"按钮
- 浏览并选择您的 Warp 应用程序文件

### 2. 配置网关设置（可选）
- **默认配置**：程序默认使用内置的网关配置
- **自定义配置**：
  - 勾选"使用自定义网关"复选框
  - 程序会验证输入的有效性

### 3. 启动代理服务
- 确认路径和网关配置正确后，点击"启动"按钮
- 程序会自动：
  - 生成必要的 SSL 证书
  - 启动本地代理服务
  - 配置网络环境

### 4. 运行 Warp
- 代理服务启动成功后
- 正常启动您的 Warp 应用
- Warp 现在应该能够正常连接和使用

### 5. 停止服务
- 使用完毕后，点击"停止"按钮
- 程序会自动清理临时配置
- 安全退出所有服务

## 📋 状态说明

软件界面会显示当前状态：

- 🟢 **"代理服务已启动"** - 服务正常运行
- 🔴 **"代理服务已停止"** - 服务未运行
- 🟡 **"正在启动..."** - 服务启动中
- ❌ **错误信息** - 显示具体的错误原因

## 🔧 自动证书管理

程序会自动处理以下证书相关操作：

1. **生成 SSL 证书**
   - 自动创建本地 CA 证书
   - 生成域名专用证书

2. **安装证书**
   - **Windows/Linux**: 自动将证书安装到系统证书存储
   - **macOS**: 需要手动执行安装命令（见上方说明）
   - 确保 SSL 连接的安全性

3. **清理证书**
   - 程序退出时自动清理临时证书
   - 保持系统环境整洁

### 🍎 macOS 证书安装详细步骤
1. 启动 Warp Infinite 程序
2. 等待程序生成证书文件（通常在 `~/certs/` 目录）
3. 打开终端，执行以下命令：
   ```bash
   sudo security add-trusted-cert -d -r trustRoot -k /Library/Keychains/System.keychain ~/certs/mitmproxy-ca-cert.pem
   ```
4. 输入管理员密码确认安装
5. 证书安装完成，可以正常使用程序

## ⚠️ 注意事项

### 使用前必读
- **关闭 VPN**：使用前必须关闭所有其他 VPN 软件
- **管理员权限**：某些操作可能需要管理员权限
- **防火墙设置**：首次运行时允许程序通过防火墙

### 常见问题
1. **Warp 无法连接**
   - 确认已关闭其他 VPN 软件
   - 检查 Warp 路径是否正确
   - 重启代理服务

2. **证书错误**
   - 程序会自动处理证书问题
   - 如有问题，重启程序即可

3. **网络异常**
   - 停止代理服务
   - 检查网络连接
   - 重新启动服务
