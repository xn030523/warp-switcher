## ❓ 常见问题

### 🔧 使用问题

**Q: 程序无法启动怎么办？**
A:
- 确认下载的版本与系统架构匹配
- 尝试以管理员权限运行
- 检查Windows Defender是否误报

**Q: 切换过程中断怎么办？**
A:
- 重新启动程序即可
- 程序会自动处理中断情况
- 不会影响原有配置

**Q: 配置没有恢复怎么办？**
A:
- 确认切换过程完全完成
- 重启WARP客户端
- 检查网络连接是否正常

### 🌐 网络问题

**Q: 提示网络连接失败？**
A:
- 检查网络连接是否正常
- 确认能够访问WARP服务
- 尝试更换网络环境

**Q: 邮箱验证失败？**
A:
- 程序会自动重试
- 耐心等待验证完成
- 避免同时运行多个实例

### 🛡️ 安全问题

**Q: 程序是否安全？**
A:
- 程序仅操作WARP相关功能
- 不收集任何个人信息
- 所有操作在本地完成

**Q: 会不会影响其他软件？**
A:
- 仅影响WARP客户端
- 不修改系统设置
- 不影响其他网络软件
